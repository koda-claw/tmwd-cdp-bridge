# tmwd-cdp-bridge — 图文介绍

> AI Agent 的浏览器 CDP 桥接

---

## 为什么需要它？

![为什么需要](tmwd-cdp-bridge-why-cn.png)

AI Agent 需要「手」来操控网页。tmwd-cdp-bridge 让任何兼容 Skill 标准的 Agent 获得浏览器控制能力 —— 执行 JavaScript、读取 DOM、拦截网络请求、截屏、操控已登录的会话。

---

## 主视觉

![主视觉](tmwd-cdp-bridge-poster-cn.png)

AI Agent（左侧）通过 CDP 桥接（中间）连接到浏览器（右侧），实现完整的浏览器自动化闭环。

---

## 开放生态

![生态兼容](tmwd-cdp-bridge-ecosystem-cn.png)

作为通用 Agent Skill 开发，任何支持本地 Skill 的 Agent 都能使用。一个桥接，统一流程。
